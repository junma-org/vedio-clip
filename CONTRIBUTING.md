# 贡献指南 | Contributing Guide

感谢您对极简视频剪辑工具的兴趣！

Thank you for your interest in Simple Video Clipper!

## 如何贡献 | How to Contribute

### 报告问题 | Reporting Issues

如果您发现了 bug 或有功能建议，请通过 [GitHub Issues](../../issues) 提交。

If you find a bug or have a feature suggestion, please submit via [GitHub Issues](../../issues).

提交问题时请包含：
- 问题的清晰描述
- 复现步骤
- 系统环境信息（Windows 版本、软件版本等）
- 相关截图（如有）

When submitting issues, please include:
- Clear description of the issue
- Steps to reproduce
- System environment info (Windows version, app version, etc.)
- Relevant screenshots (if applicable)

### 提交代码 | Submitting Code

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

---

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

### 代码规范 | Code Style

- 遵循 PEP 8 Python 代码风格指南
- 使用有意义的变量名和函数名
- 添加适当的注释和文档字符串
- 保持代码简洁可读

---

- Follow PEP 8 Python style guide
- Use meaningful variable and function names
- Add appropriate comments and docstrings
- Keep code clean and readable

## 开发环境设置 | Development Environment Setup

```bash
# 克隆仓库 | Clone repository
git clone https://github.com/yourusername/video-clipper.git
cd video-clipper

# 创建虚拟环境（推荐）| Create virtual environment (recommended)
python -m venv venv
venv\Scripts\activate  # Windows

# 安装依赖 | Install dependencies
pip install -r requirements.txt

# 运行 | Run
python main.py
```
