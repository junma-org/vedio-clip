# Bundled Fonts

Place subtitle font files here before packaging.

Supported extensions:

- `.ttf`
- `.otf`
- `.ttc`

The app loads files from this directory at startup, and `VideoClipper.spec` includes this directory in PyInstaller builds. Recommended Chinese subtitle fonts include Source Han Sans SC, Noto Sans CJK SC, HarmonyOS Sans SC, LXGW WenKai, and Smiley Sans.
