# v0.10 字幕编辑交互优化

日期：2026-04-29

## 需求

- 修复选中字幕后勾选“渐隐渐显”并设置时长，点击更新后没有写入字幕效果的问题。
- 支持对单条或多条选中字幕批量设置渐隐渐显、字体、字号、颜色和字幕模板。
- 字幕参数修改应即时应用，不再依赖“更新”按钮；模板切换也应即时应用，不再依赖“应用”按钮。
- 字幕列表支持常规快捷键：`Ctrl+A` 全选、`Delete` 删除选中字幕、复制选中字幕到外部文本时输出 SRT 格式。
- 移除字幕区“删除”“清空”“更新”和模板“应用”按钮，降低操作负担。
- “短视频口播”模板位置下移约 8%，字号从 66 调小 6 号。

## 实现口径

- 保持内部字幕工程模型为 ASS 风格，SRT 导入继续转换为 ASS 项目用于样式处理。
- 即时编辑只作用于当前选中的字幕行；多选时样式类控件批量作用于所有选中字幕。
- 文本和时间编辑保留单条字幕语义，避免多选时把多条字幕的文本或时间误改成同一个值。
- 复制到剪贴板输出 SRT 文本，只改变外部复制格式，不影响导入和导出烧录流程。
- GUI 只做交互状态收集，ASS 标签仍通过 `subtitle_model.py` 的工具函数处理。
- 单选或部分多选修改字体、字号、颜色时，如目标样式名还被未选中字幕使用，则生成私有样式，避免同名 ASS 样式的全局影响误伤未选中字幕。

## 完成状态

## 风险

- GUI 交互依赖 PySide6，本轮只能覆盖纯模型测试和语法检查，需要在 Windows 打包环境点验多选、复制、删除和预览渐隐渐显。
- 部分选中产生的私有样式会保留在项目样式列表中，后续如需要可再做未使用样式清理。
