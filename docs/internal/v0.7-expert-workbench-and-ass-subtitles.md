# v0.7 达人编辑台与 ASS 字幕

## 需求

- 达人模式改成专用编辑台，不再复用顶部拖入区和 MVP 表单式布局。
- 切换到达人模式时自动最大化窗口；返回极简模式时恢复原窗口状态。
- 达人模式围绕视频预览、播放头、时间轴和选区操作，不再要求用户手动设置入点/出点。
- 支持基于选区删除片段、基于播放头删除当前帧、基于选区或播放头插入字幕。
- 字幕统一按 ASS 内部模型处理与导出，支持导入 ASS/SRT 文件，也支持粘贴 ASS/SRT 文本。
- SRT 导入或粘贴后自动转换为 ASS 内部模型；导出统一烧录 ASS 硬字幕。
- 提供字幕列表，支持快速编辑文本、时间、模板样式和删除。
- 播放预览时要能实时看到当前字幕的大致效果。
- 修复 Windows 上 FFmpeg 输出读取导致的 `gbk` 解码异常；极简模式和达人模式都要走同一套稳定处理链。

## 假设和取舍

- 本版本只做单视频轨、单时间轴选区，不做多轨、波形、关键帧、撤销重做。
- 实时字幕预览追求模板样式、位置和文字内容可见；对外部 ASS 的复杂 override tags 只保证导出保真，不承诺预览逐像素一致。
- 外部 ASS 导入采用“混合保真”：
  - 时间、样式名、样式定义和原始事件文本尽量保留；
  - 编辑器中允许改文本和时间；
  - 若用户改写文本，复杂内联标签可能退化为保留前导标签。
- 导出时 FFmpeg 统一使用 ASS 临时文件；GUI 不直接拼字幕 filter 文本。
- FFmpeg 进度改为解析 `-progress pipe:1` 机器输出，避免再依赖本机默认编码去读命令行日志。

## 技术方案

- 新增 `timeline_state.py`：
  - 表达播放头、选区、删除当前帧、选区转删除区间、无选区时按默认 2 秒插入字幕的纯逻辑。
- 重做 `subtitle_model.py`：
  - 用 `SubtitleCue`、`SubtitleStyleDef`、`SubtitleProject` 表达 ASS 字幕工程。
  - 支持 ASS/SRT 文件导入、文本粘贴导入、ASS 写出、样式模板和按时间查询当前字幕。
- 扩展 `edit_model.py`：
  - `EditPlan.subtitles` 升级为 `SubtitleProject`，其余导出入口保持不变。
- 重做 `ffmpeg_utils.py`：
  - 用统一的字节流读取和安全解码处理 FFmpeg/FFprobe 输出。
  - 新增 `run_ffmpeg_with_progress()`，解析 `out_time_ms` / `out_time_us` 进度。
  - 把字幕临时文件从 SRT 改为 ASS，并保持 filter 顺序为 `subtitles -> select/setpts -> scale`。
- 重做 `gui.py`：
  - 主界面改成 `QStackedWidget`，分为极简模式页和达人编辑台页。
  - 达人编辑台使用懒加载播放器；无视频时显示空态卡片，加载后再显示视频预览和时间轴。
  - 预览层使用图形场景视频项与字幕叠加绘制，实现实时字幕近似预览。
  - 新增自定义时间轴控件，显示删除块、字幕块、播放头和选区。
  - 新增字幕侧栏：文本编辑、模板切换、导入文件、粘贴文本、列表编辑、删除。
- 引入 `pysubs2` 作为 ASS/SRT 读写和转换主路径；内部模型保留为 GUI/导出稳定接口，并在缺少依赖的极简测试环境中提供兜底。
