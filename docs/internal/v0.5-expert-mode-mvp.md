# v0.5 达人模式 MVP

## 需求

- 新增达人模式 MVP，能力只包含视频预览、定位时间点、设置入点/出点、加入删除区间、删除已加入区间、导出。
- 达人模式导出必须复用 v0.4 的 `EditPlan`，不要绕过统一编辑模型。
- 极简模式继续保留现有表单能力，不因为达人模式引入字幕、特效、完整时间轴、关键帧或复杂剪映式界面。
- FFmpeg 命令构造继续集中在 `ffmpeg_utils.py`，GUI 只负责收集参数、展示状态和轻校验。

## 假设和取舍

- 视频预览使用 PySide6 自带 Qt Multimedia 控件；它负责播放、暂停和拖动定位，不新增外部播放器依赖。
- 当前版本只记录删除区间，不做保留区间、撤销重做、逐帧步进或波形/缩略图时间轴。
- 入点和出点以当前播放位置为准，加入删除区间时再做范围校验和归一化。
- 达人模式导出本次不新增字幕、特效和模板参数，避免提前扩大 `EditPlan` 的职责。

## 技术方案

- 在 `gui.py` 中增加模式页签：
  - 极简模式承载现有拖入、剪开头、删除秒数区间和分辨率表单。
  - 达人模式承载视频预览、播放/暂停、进度滑块、秒数定位、入点/出点按钮和删除区间列表。
- 选中文件后同时更新极简模式第一帧预览和达人模式媒体源。
- 新增轻量纯逻辑辅助，负责达人模式入点/出点转删除区间，便于在无 GUI 环境下测试。
- `start_processing()` 根据当前模式生成 `EditPlan`：
  - 极简模式沿用现有 `build_edit_plan_from_controls()`。
  - 达人模式使用删除区间列表生成 `EditPlan` 并复用现有导出线程。

- 自动化测试：`python3 -m unittest discover -s tests`。

- 手动验证：
  - 选择视频后达人模式显示预览控件并可播放/暂停。
  - 拖动或输入秒数后能定位时间点。
  - 设置入点/出点后能加入删除区间，区间列表可删除和清空。
  - 达人模式导出成功生成 MP4，删除区间生效。

## 遗留风险和下一步

- 需要在安装了 `PySide6` 和 FFmpeg 的 Windows 打包环境中手动验证 Qt Multimedia 播放、拖动定位和导出效果。
- Qt Multimedia 的格式支持受本机解码环境影响；若 Windows 打包版预览兼容性不足，再考虑 FFmpeg 抽帧预览兜底，不在 v0.5 扩大范围。
